# 🚗 Информационная система: CarSpecs App

Современное десктопное приложение-справочник, которое в реальном времени подтягивает технические характеристики автомобилей из внешней базы данных через REST API, обрабатывает полученный JSON и генерирует структурированные отчеты[cite: 10, 11, 12].

## 🛠 Технологический стек
* **Язык разработки:** Python 3[cite: 12]
* **Графический интерфейс (GUI):** CustomTkinter (динамическая поддержка светлой и темной тем оформления)[cite: 11, 12]
* **Сетевые запросы:** Библиотека `requests` для синхронизации с API по протоколу HTTP[cite: 10, 11, 12]

## 🎯 Ключевые возможности приложения
* **Многоуровневая фильтрация:** Интерактивные выпадающие списки (марка -> модель -> комплектация) динамически блокируются и обновляются в зависимости от шага пользователя, исключая отправку некорректных запросов[cite: 11].
* **Глубокая интеграция с API:** Программа последовательно работает с тремя эндпоинтами CarAPI (загрузка марок, фильтрация моделей v2 и получение ТТХ комплектаций v2)[cite: 10, 11].
* **Экспорт отчетов (I/O операции):** Реализована функция сохранения сгенерированного технического паспорта автомобиля в локальный файл `.txt` с автоматическим формированием понятного имени файла[cite: 11].
* **Отказоустойчивость:** Сетевой клиент защищен таймаутами и блоками `try-except` для перехвата ошибок при отсутствии интернета или сбоях сервера[cite: 10, 11].

## 📂 Структура кодовой базы
Проект спроектирован по модульному принципу для разделения логики сети и интерфейса[cite: 10, 11]:
* `main.py` — графический интерфейс информационной системы, обработчики событий элементов UI и логика файлового вывода отчетов[cite: 11].
* `api_client.py` — изолированный сетевой клиент, отвечающий за формирование URL, заголовков и обработку JSON-ответов сервера[cite: 10].
* `.gitignore` — конфигурация исключений для предотвращения отслеживания кэша `__pycache__` и локальных настроек IDE[cite: 13].
